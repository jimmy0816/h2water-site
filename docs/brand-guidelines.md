## 語氣規則

### ✅ Do
- 用「根據…研究」「研究結果顯示」等有依據的說法
- 主動標示科學證據等級（強/初步/不足/無效）
- 引用論文時附上期刊名稱、年份、DOI
- 對不確定的說法用「初步研究顯示，但尚需更多確認」
- 每篇結尾附上免責聲明（非醫療建議）
- 提供具體數據：ppm 濃度、受試人數、研究持續時間

### ❌ Don't
- 不使用「治療」「療效」「根治」等醫療宣稱詞彙
- 不說「最有效」「第一名」（除非有直接比較研究）
- 不貶低其他健康飲品或產品
- 不捏造或誇大研究數據
- 不在無研究支持的情況下推薦特定品牌

## Evidence Badge 規格

本站核心功能，每篇文章**必須標示**其主要論點的科學證據等級：

| 等級 | Badge | 標準 |
|------|-------|------|
| 強證據 | ✅ 強證據 | 多個 RCT 或系統性回顧，結果可重複 |
| 初步證據 | 🔬 初步證據 | 1-2 個 RCT 或觀察性研究，待確認 |
| 證據不足 | ⚠️ 證據不足 | 動物實驗或小樣本（n < 20），尚無大型人體研究 |
| 無效/誤導 | ❌ 無效/誤導 | 研究顯示無效，或與科學共識相悖 |

**使用原則**：
- 每篇文章必須至少有一個 Evidence Badge
- Badge 放在文章標題下方、Description 前面
- 內文中也可對特定段落加 inline Badge
- 不確定時，選更保守的等級

## Citation Card 規格

論文引用必須使用 Citation Card 特殊樣式：

```html
<div class="citation-card">
  <div class="citation-journal">JOURNAL NAME · YEAR</div>
  <p class="citation-conclusion">研究主要結論（1-2 句）</p>
  <a href="doi-url" class="citation-doi">DOI: xx.xxxx/xxxxx →</a>
</div>
```

**使用原則**：
- 每次直接引用研究數據時都要附 Citation Card
- 期刊名稱用縮寫大寫（如 Sci Rep, PLOS ONE, J Clin Med）
- 結論要用自己的話重新描述，不直接翻譯原文
- 必須附上可點擊的 DOI 連結

## SEO 內容策略

### 結構化資料
- 每頁嵌入 JSON-LD：`MedicalWebPage` / `Article` / `FAQ`
- 文章頁加 `datePublished`、`dateModified`、`author`

### 關鍵字策略
| 主題 | 目標關鍵字 |
|------|-----------|
| 氫水科學 | 氫水功效、氫水真的有效嗎、氫水科學研究 |
| 研究文獻 | 氫水論文、氫水研究、hydrogen water study |
| 健康應用 | 氫水抗氧化、氫水運動、氫水發炎 |

### Core Web Vitals 目標
- LCP ≤ 2.5s（Hero 圖片 preload）
- CLS ≤ 0.1（圖片標示寬高，字型 display=swap）
- INP ≤ 200ms

## i18n 預留規範

- 台灣繁體中文為主（`lang="zh-TW"`）
- HTML `lang` 屬性必須正確設定
- 文章 frontmatter 保留 `lang: 'zh' | 'en' | 'ja'` 欄位
- 未來日文版 URL 結構：`/ja/blog/[slug]/`
- 勿硬編碼台灣特定內容（如「台灣食藥署」）而應設計為可替換
